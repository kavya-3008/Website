# Kavya Shah Portfolio Website

A sleek, minimal multi-page portfolio site using your resume content.

## Pages

- `index.html` — Home page with intro, highlights, and profile image block.
- `work.html` — Selected projects and technical outcomes.
- `experience.html` — Education, roles, and skill summary.
- `contact.html` — Contact details and links.

## Run locally

```bash
python3 -m http.server 4173
```

Open <http://localhost:4173>.

## Personalize in 5 minutes

1. Replace `assets/kavya-profile-placeholder.svg` with your real photo.
2. Update the short intro paragraph in `index.html`.
3. Add links to GitHub/project demos in `work.html`.
4. Add downloadable resume link in `contact.html`.
5. Deploy to Netlify, Vercel, or GitHub Pages.
