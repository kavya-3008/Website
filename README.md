# Kavya Shah Portfolio

A sleek, minimal one-page portfolio inspired by clean editorial-style personal websites.

## Run locally

```bash
python3 -m http.server 4173
```

Then open: <http://localhost:4173>

## Quick customization

1. Replace project cards in `index.html` with your real work.
2. Update the contact links and email.
3. Adjust colors in `styles.css` via `:root` variables.
4. Add deployment via GitHub Pages, Netlify, or Vercel.
